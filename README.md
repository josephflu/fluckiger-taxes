# Fluckiger Taxes

Personal tax document repository for Joseph & Sarah Fluckiger. Filing status: Married Filing Jointly.

## Workflow

Tax documents (PDFs) are organized by source in `taxes/<year>/`. Each PDF is processed and gets a matching `.md` file with extracted tax-relevant data. The document inventory and tax estimate live in:

- `Taxdocs_2025_Doc_Inventory.csv` — tracks every document with status, key amounts, and file paths
- `TAX_SUMMARY_2025.md` — income, deductions, credits, and estimated tax calculation
